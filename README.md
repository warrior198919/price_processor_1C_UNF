# Обработчик прайс-листов → 1С УНФ

Программа для автоматической конвертации прайс-листов **SATA** и **RoxelPro**
из Excel (.xlsx) в формат, совместимый с загрузкой в **1С:УНФ** (Управление нашей фирмой).

---

## 📁 Структура проекта

```
price_processor/
├── main.py                  ← GUI-версия (с графическим интерфейсом)
├── cli.py                   ← Консольная версия
├── parsers/
│   ├── sata_parser.py       ← Парсер прайса SATA
│   └── roxelpro_parser.py   ← Парсер прайса RoxelPro
└── exporters/
    └── unf_exporter.py      ← Экспорт в формат 1С УНФ
```

## 🚀 Использование

### Вариант A — Графический интерфейс
```bash
python main.py
```

![GUI]  
1. Нажмите **➕ Добавить файлы** — выберите XLSX прайсы  
2. Выберите **Формат прайса** (auto / SATA / RoxelPro)  
3. Выберите **папку для сохранения**  
4. Нажмите **▶ Обработать**  

### Вариант B — Командная строка
```bash
# Одиночный файл (автоопределение бренда)
python cli.py SATA.xlsx

# Несколько файлов
python cli.py SATA.xlsx RoxelPro-Foto.xlsx --out ./output

# Явный бренд, цена в у.е., разбить по группам
python cli.py SATA.xlsx --brand SATA --price ue --split

# Без фото-ссылок
python cli.py RoxelPro-Foto.xlsx --no-photos
```

**Все параметры CLI:**
| Параметр | Значение | По умолчанию |
|---|---|---|
| `--brand` | auto / SATA / RoxelPro | auto |
| `--out` | папка сохранения | текущая |
| `--price` | rub / ue | rub |
| `--split` | разбить по группам | нет |
| `--no-photos` | убрать ссылки на фото | нет |

## 📊 Формат результата (1С УНФ)

Создаётся файл `1С_УНФ_SATA_YYYYMMDD_HHMMSS.xlsx` с двумя листами:

### Лист «Номенклатура»
| Колонка | Описание |
|---|---|
| Наименование | Полное название товара |
| Артикул | Артикул поставщика |
| Единица измерения | шт., уп., кор. |
| Ставка НДС | 20% |
| Цена продажи (руб) | Розничная цена |
| Описание | Описание модели |
| Группа номенклатуры | Путь в дереве (Бренд\Группа) |
| Бренд (производитель) | SATA / RoxelPro |
| Штрихкод | Штрихкод (если есть) |
| Изображение | URL фотографии |
| Полное наименование | Дублирует Наименование |
| Vendorcode | Код поставщика |
| Подгруппа | Категория товара |

### Лист «Инструкция»
Пошаговая инструкция по загрузке данных в 1С УНФ.

## 🔄 Загрузка в 1С УНФ

1. Меню **Данные** → **Загрузка данных из файла**
2. Укажите путь к созданному файлу
3. Выберите лист **Номенклатура**
4. Сопоставьте колонки (автоматически по названиям)
5. Выберите режим: **Создавать новые** или **Обновлять существующие**
6. Нажмите **Загрузить**

Создайте файл `parsers/mybrand_parser.py`:

```python
class MyBrandParser:
    def __init__(self, path: str):
        self.path = path

    def parse(self) -> list[dict]:
        # Верните список словарей с ключами:
        # Номенклатура, Артикул, Описание, Количество,
        # Цена розничная (руб), Ед. изм., Валюта, Бренд,

        return []
```

Затем добавьте в `main.py` и `cli.py` ветку для нового бренда.

## 📝 Особенности парсинга SATA

Прайс SATA имеет сложную иерархическую структуру:
- Каждый **товар-модель** содержит несколько **вариантов** по размеру дюзы (1,1 I / 1,2 I / 1,1 O и т.д.)
- Каждый вариант имеет **отдельный артикул**
- Программа разворачивает каждый вариант в **отдельную строку** номенклатуры

Например, модель `SATAjet X RP BASIC` → 8 строк с артикулами 1200295, 1200378, ...
